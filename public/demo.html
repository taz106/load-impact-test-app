JS component usage

There are 2 components

  1. SelectWidget
  2. UrlList

SelectWidget Component

Description: It is form component which has 2 fields. Select category and select sub-category. And selecting both of them is required. After selecting both you have to press the submit button you can view the url list. There is validation message for trying to click the button before selecting both of them.

Url List

Description: It is list component with every url as the list item. you select any url from the list.
